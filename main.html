Project 1 : Monkeypox Classification

Developed a machine learning pipeline to accurately classify monkeypox skin lesions using medical image data.
  • Dataset: Monkeypox Skin Lesion Dataset from Kaggle (original + augmented images).
  • Challenge Tackled: Addressed severe class imbalance using ADASYN (Adaptive Synthetic Sampling) to generate synthetic samples for minority classes.
  • Approach:
    • Extracted discriminative features with ResNet50 (pre-trained CNN).
    • Trained a Support Vector Machine (SVM) with a polynomial kernel for robust classification.
  • Results:
    Achieved 97.7% accuracy on the test set with high precision, recall, and F1-scores across all classes.

    Project 3 : Emotion Classification via Face Landmarks

    Developed a computer vision system to recognize human emotions (happy vs. sad) using facial landmarks.
    • Dataset: AffectNet Happy-Sad subset (facial images labeled with positive/negative emotions).
    • Feature Extraction: Leveraged MediaPipe Face Mesh to capture 468 3D facial landmarks per face.
    • Preprocessing: Train-test split, shuffling, and SMOTE oversampling to handle class imbalance.
    • Models:
      • Deep Neural Network (DNN): Multi-layer perceptron with dropout + hyperparameter tuning.
      • Ensemble Methods: Random Forest & Extra Trees with feature importance analysis.
      • SVM: Tested multiple kernels (linear, RBF, polynomial).
    • Evaluation: Accuracy, confusion matrix, ROC-AUC, and cross-validation for robust performance.
    • Results: Achieved 84% accuracy in emotion classification.
    • Impact: Demonstrated real-time emotion recognition capabilities through landmark-based analysis, with applications in affective computing and HCI.

    Project 4 : Depression Detection from Text using NLP & Deep Learning

    • Dataset: Custom dataset (binary classes: Depression vs. Normal).
    • Text Preprocessing: Tokenization, stopword removal, lemmatization with NLTK.

    • Evaluation: Classification metrics, confusion matrix, ROC-AUC, and cross-validation.

    Project 5 : EEG Depression Classification

    Implemented a machine learning pipeline for emotion recognition (positive vs. negative valence) using EEG signals from the DEAP dataset.
    • Dataset: DEAP (32 EEG channels, 128 Hz sampling, video-watching trials with self-assessment labels).
    • Signal Processing: Applied Short-Time Fourier Transform (STFT) for time-frequency decomposition.
    • Created Enhanced Frequency Domain Maps (EFDM) by mapping electrode signals onto spatial-spectral grids.
    • Feature Engineering: Extracted Histogram of Oriented Gradients (HOG) features from EFDM representations.
    • Dimensionality Reduction: Applied PCA and scaling for dimensionality reduction and feature optimization.
    • Models: Random Forest, Extra Trees, and LightGBM with hyperparameter tuning.
    • Imbalanced Learning: Addressed dataset imbalance using Borderline SMOTE.
    • Evaluation: Conducted cross-validation, confusion matrix analysis, and ROC-AUC assessment.
    • Results: Demonstrated a novel EEG-to-image approach that effectively classifies emotions via spatial-spectral representations.
